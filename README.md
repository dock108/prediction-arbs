# arbscan

A cross-venue sports betting mis-pricing scanner.

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/arbscan.git
    cd arbscan
    ```

2.  **Install dependencies:**
    ```bash
    poetry install
    ```

3.  **Run tests:**
    ```bash
    poetry run pytest
    ```
